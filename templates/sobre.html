Este sistema foi desenvolvido Por Ivonildo Lima com a finalidade de facilitar ávida nas operações com a Ucaq. É uma aplicação web desenvolvida com as tecnologias Flask, HTML, Javascript, Python, SQLite como banco, que permite o cálculo de volumes com base em distâncias e opções predefinidas. Os resultados dos cálculos são armazenados em um banco de dados SQLite, com funcionalidades para visualização e exclusão dos registros. O sistema possui uma interface para inserir dados, como a cidade e o poço, calcular o volume e salvar os resultados. Ele também mantém um histórico dos últimos 15 resultados e permite que os usuários gerenciem esses registros diretamente pela interface do site. Além disso, o sistema utiliza mensagens flash para fornecer feedback ao usuário sobre as operações realizadas.
